DADOS PESSOAIS

TIPOS DE DADOS PESSOAIS

Esta seção é dedicada ao cadastro do TIPO de um dado pessoal. Por "tipo", classifica-se a generalização do dado: RG, CPF, EMAIL, NOME, etc.

FONTES DE DADOS

TIPOS DE FONTES DE DADOS

Em resumo, o driver de conexão para uma fonte de dados do tipo Banco de Dados. Para que a máquina possa conectar-se a um Banco de Dados, primeiro é necessário especificar COMO a máquina deve comunicar-se com este Banco de Dados.

BANCO DE DADOS

A seção de Bancos de Dados registra os dados de acesso para um determinado banco de dados que será, mais tarde, varrido pelo algoritmo.

ARQUIVO DE DADOS

A seção de Arquivo de Dados serve para registrar e procurar dados específicos dentro de diretórios e arquivos.
