BASES LEGAIS E FINALIDADES

BASES LEGAIS

As bases legais da LGPD legitimam a utilização de dados pessoais. Cada operação de tratamento de dados pessoais deve ter ao menos uma base legal vinculada.

FINALIDADES

As finalidades são os propósitos determinados a tal tratamento de dados. Realize o cadastro de uma finalidade específica e então adicione uma "Finalidade por Base Legal".

FINALIDADE POR BASE LEGAL

Uma finalidade deve ser atrelada a uma base legal. A base legal é o que define as políticas de tratamento de dados; a finalidade é a especificação de "para o quê" o dado é utilizado.
