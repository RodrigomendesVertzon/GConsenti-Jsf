DATA LEARNING

APRENDIZADO

EXPRESSÃO REGULAR

Expressões Regulares são padrões simples que categorizam dados de acordo com o formato do dado. Se um tipo de dado possui um padrão simples e observável, então o melhor método de identificação é uma expressão regular.

SCRIPTING

Script é uma série de comandos executados linha a linha que têm como função definir dados que não podem ser categorizados por expressões regulares.

INTELIGÊNCIA ARTIFICIAL

Este tipo de análise é recomendada quando não há padrão em um tipo de dado. Nomes pessoais ou endereços, como um exemplo, não mantém um padrão lógico além de seu padrão linguístico. Como uma máquina não pode entender uma língua, uma análise estatística é necessária.
